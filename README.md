# Yara Generator

YARA rule generator.

Generator aturan YARA.


## Description

Automatically creates YARA detection rules by analyzing file patterns and extracting unique strings.

## Deskripsi

Secara otomatis membuat aturan deteksi YARA dengan menganalisis pola file dan mengekstrak string unik.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Generate: python yara_gen.py malware.exe
- Batch: python yara_gen.py samples/ -o rules.yar
- Demo: python yara_gen.py --demo

## Penggunaan

- Generate: python yara_gen.py malware.exe
- Batch: python yara_gen.py samples/ -o rules.yar
- Demo: python yara_gen.py --demo
